Block Definition

HCWidgetOpenProfileBlock

The widget contains a well designed, out of box user profile page. However, if your app also has an user profile page, you can utiltize this callback to link the widget’s profile page and your profile page together. When you initialize the widget, you can enable the widget by setting true to kHCDefaultSocialWidgetEnableProfileLink. You can ignore this if your app doesn’t have a profile page. When the user clicks on the profile linking button in widget profile page, the callback block will happen, and you can take this opportunity to open the profile page.
