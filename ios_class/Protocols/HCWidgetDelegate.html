– shouldOpenAppUserProfilePage

If your app has user profile page and you don’t want to use the profile page we provide with the widget, please implement this method in the delegate and return YES. Otherwise, return NO, or simply do not implement this method.

- (BOOL)shouldOpenAppUserProfilePage

Return Value

TRUE if you don’t want to open widget’s user profile page.

Declared In

HCWidget.h

– openProfileWithUserID:

If your app has user profile page and you don’t want to use the profile page we provide with the widget, when a user wants to open profile inside the widget, the widget can tell your app, so you can close the widget and go to this user’s profile. If this is the behavior you wanted, you should also implement shouldOpenAppUserProfilePage and return TRUE.

- (void)openProfileWithUserID:(NSString *)userID
